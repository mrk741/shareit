How This Works

  1. Sender: Select or drop a file. A special "magnet link" is generated.
  2. Sender: Copy this link and send it to the receiver (via chat, email, etc.).
  3. Receiver: Paste the link into the "Receive" box and click "Start Download".
  4. The file is transferred directly from the sender's browser to the receiver's browser.

IMPORTANT: The sender must keep this browser tab open until the receiver has finished downloading. If the sender closes the tab, the transfer will stop.
